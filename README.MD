# Resource Allocation System (Java)

## Description
A console-based system to manage employees and task allocation.

## Features
- Task assignment with constraints
- Prevent employee overload
- Search tasks
- View analytics (stats)
- Filter tasks by employee

## Tech Used
- Java
- OOP Concepts
- HashMap (DSA)

## Future Improvements
- Convert to REST API (Spring Boot)
- Add database
- Add frontend (React)
